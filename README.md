# 🎵 Vibe Learning Companion

Aplikasi pendamping belajar untuk *vibe coding* yang minimalis dan efisien.

## 🚀 Fitur Utama
- **Habit Tracker**: Bangun kebiasaan belajar konsisten dengan visualisasi progress.
- **Pomodoro Timer**: Fokus dalam sesi 25 menit dengan auto-switch ke istirahat 5 menit.
- **Telegram Quick Capture**: Integrasi bot Telegram untuk catatan cepat & daily report.

## 🔗 Live App
[https://ahmadbb1992.github.io/vibe-learning-companion/](https://ahmadbb1992.github.io/vibe-learning-companion/)

## 🛠️ Tech Stack
- HTML5, CSS3 (Dark Mode), Vanilla JavaScript
- localStorage (data persistence)
- Telegram Bot API
